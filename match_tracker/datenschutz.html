Datenschutzerklärung
1. Allgemeine Hinweise

Diese Website ist ein privates Tool zur Verwaltung von Spiel-, Team- und Statistikdaten (Match Tracker). Der Schutz deiner Daten ist uns wichtig. Personenbezogene Daten werden nur im technisch notwendigen Umfang verarbeitet.

2. Verantwortlicher

Verantwortlich für die Datenverarbeitung auf dieser Website ist der Betreiber dieses Projekts.
E-Mail: kontakt@d2r-lootliste.de

3. Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Seite werden automatisch technische Daten erfasst (z. B. IP-Adresse, Datum und Uhrzeit, Browserinformationen).

4. Lokale Datenspeicherung (LocalStorage)

Die von dir eingegebenen Daten (z. B. Teams, Spieler, Spiele, Statistiken) werden lokal im Browser gespeichert (LocalStorage). Diese Daten verbleiben auf deinem Gerät und werden nicht automatisch an Dritte übertragen.

5. Server-Speicherung und Synchronisation

Zur optionalen Synchronisation können Daten an einen Server übertragen und dort gespeichert werden.
Server: https://rare-kindness-production.up.railway.app

Dabei können folgende Daten verarbeitet werden:
  • Spiel- und Teamdaten
  • technische Metadaten (z. B. IP-Adresse, Zeitstempel)
6. Zugriffsschutz (Passwort-Funktion)

Einzelne Teams können durch ein Passwort geschützt werden. Die Eingabe erfolgt clientseitig und wird zur Überprüfung an den Server übertragen.

7. Keine Weitergabe an Dritte

Es erfolgt keine Weitergabe deiner Daten an Dritte.

8. Keine Tracking-Tools

Es werden keine Tracking- oder Analyse-Tools eingesetzt.

9. Rechtsgrundlage

Die Verarbeitung erfolgt gemäß Art. 6 Abs. 1 lit. f DSGVO.

10. Deine Rechte

Du hast das Recht auf Auskunft, Berichtigung oder Löschung deiner Daten.
